## FarmLearn
FarmLearn is a beginner-friendly online farming learning and management platform designed to help individuals start farming with zero prior knowledge. 
The system provides structured guidance, categorized learning content, scheduling tools, and interactive features to support new farmers.
## Features
- Categorized farming learning resources
- Step-by-step crop guidance
- Farming activity calendar & scheduling
- AI-based chatbot for farming assistance
- Secure user authentication (Login/Register)
- Responsive design for all devices
## Tech Stack
- React (Vite)
- HTML5
- CSS3
- JavaScript
- npm (Node Package Manager)

## Installation 
1. Clone the repository
git clone https://github.com/srividya-1638/FarmLearn.git
2. Navigate into the project folder
cd FarmLearn
3. Install dependencies
npm install
4. Run the development server
npm run dev
The application will start locally (usually at):
http://localhost:8080
## Purpose of the Project
Many beginners rely on scattered and unstructured online resources. 
FarmLearn aims to provide a centralized, organized, and interactive learning platform combining education, scheduling, and real-time assistance in one system.
